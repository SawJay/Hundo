CountUP!: A Coding Challenge

Living life one loop at a time. CountUP! is a JavaScript coding challenge using loops. The objective is to write a program which displays the numbers 0 to 100 with the even numbers in Bold text.

Instructions

Enter a starting value and ending value. Press the button and watch the values between those numbers appear on the page!
